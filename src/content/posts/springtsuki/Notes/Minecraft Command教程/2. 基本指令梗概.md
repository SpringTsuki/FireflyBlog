# 2. 基本指令梗概（Java 1.21.5 版本为例）

```
本页面的 wiki 链接如下：
https://zh.minecraft.wiki/w/%E5%91%BD%E4%BB%A4?variant=zh-cn
```

**命令（Command），**也被称作**控制台命令（Console Command）**或**斜杠命令（Slash Command）**，是通过输入特定文本字符串使用的高级功能。

这里的话就介绍一些相对于简单的指令（attribute、execute、data 等后续会拎出来单独教学）

指令大全：

## 1. 游戏内指令

```
/advancement # 进度
/difficulty # 游戏难度
/clone # 复制
/damage # 对实体造成伤害
/effect # 实体状态
/enchant # 添加魔咒（受限于铁砧机制）
/experience & /xp # 经验
/fill # 填充
/fillbiome # 设置指定区域的生物群系
/gamemode # 设置玩家游戏模式
/gamerule # 设置游戏规则
/give # 给予玩家物品
/help # 提供命令帮助
/kill # 清除实体
/locate # 定位结构
/playsound # 播放声音
/ride # 控制实体骑行状态
/rotate # 更改实体旋转角度
/say # 说话
/seed # 地图种子
/setblock # 放置方块
/setworldspawn # 设置世界重生点
/spawnpoint # 设置重生点
/spectate # 使一个处于旁观模式的玩家旁观一个实体
/stopsound # 停止声音
/teleport # 传送
/tell # /msg & /w的替代命令，向另一个玩家发送私信
/teammsg # 队伍消息
/tick # 停止或开始游戏刻的流逝，或更改游戏刻速率
/time # 更改世界时间
/tm # /teammsg的替代命令
/tp # /teleport的替代命令
/w # /msg & /tell的替代命令，向另一玩家发送私信
/weather # 天气
/worldborder # 设置世界边界
/xp # /experience的替代命令
```

## 2. 高级指令

```
/attribute # 属性
/bossbar # Boss栏
/clear # 清除
/data # 获取、合并、修改或移除方块、实体或命令存储的NBT数据。
/execute # 执行（超级复杂）
/forceload # 强制区块加载
/function # 执行函数
/item # 修改方块（箱子、熔炉等）或实体（玩家或生物）物品栏内的物品。
/loot # 战利品
/particle # 创建粒子
/place # 放置结构
/random # 抽取随机值
/recipe # 给予或剥夺合成配方
/return # 返回
/schedule # 安排函数在特定的游戏刻后运行
/scoreboard # 计分板
/spreadplayers # 随机传送
/summon # 生成生物
/tag # 标签
/team # 队伍
/tellraw # 向玩家发送以文本组件表示的消息
/title # 标题
/trigger # 设置一个触发器
```

## 3. 服务器指令

```
/ban # 封禁
/ban-ip # 封禁IP
/banlist # 封禁列表
/defaultgamemode # 默认游戏模式
/deop # 解除OP
/kick # 踢出玩家
/list # 列出服务器的玩家
/me # 显示一条关于自己的信息
/msg # /tell & /w 的替代命令，向另一玩家发送私信。
/op # 给予OP
/pardon # 从封禁列表中移除玩家
/pardon-ip # 从封禁列表中移除IP
/publish # 向局域网开放单人游戏世界
/reload # 重载
/save-all # 保存所有数据
/save-off # 保存数据并关闭服务器
/save-on # 保存数据并开启服务器
/setidletimeout # 设置无操作玩家被踢出的延时
/stop # 关服
/transfer # 将玩家转移到另一个服务器上
/whitelist # 白名单
```

## 4. 开发指令

```
/datapack # 数据包
/debug # 调试
/jfr # JFR分析
/perf # 创建性能分析
/test # 测试
/warden_spawn_tracker # 设置幽匿尖啸体生成监守者之前应接收到多少次振动(仅IDE环境)
```
