# SELECT 与 SELECT DISTINCE 语句

#### 1. SELECT 语句

SELECT 用于从数据库中选取数据，用法如下：

```sql
SELECT column1, column2 FROM table_name;
```

这里 col1 与 col2 就是从表中要查询的 **列** 的字段名称



```sql
SELECT * FROM table_name
```

从表中选取所有的列



#### 2. SELECT DISTINCE 语句

```sql
SELECT DISTINCT column1, column2 FROM table_name;
```

有时，一个表中的列中可能存在相同的数据，例如一个名为 country 中的列，ID为1的行，此值为CN，以此类推，2为CN，3为US，4为JP

那么使用以下查询，则结果只会显示为CN、US与JP

```sql
SELECT DISTINCT country FROM table;
```

结果为：

```sql
country
CN
US
JP
```
