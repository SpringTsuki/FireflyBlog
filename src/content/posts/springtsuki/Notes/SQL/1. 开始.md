#### 1. Docker 部署 MySQL

详情请看 Docker 部分的 第三部分（Dockerfile 详解）

部署完毕后，记得需要修改 root 的权限，让它可以通过远程访问。顺便给 root 改个密码吧

```sql
GRANT ALL PRIVILEGES ON *.* TO 'root'@'%' WITH GRANT OPTION;
ALTER USER 'root'@'%' IDENTIFIED BY '132120';
FLUSH PRIVILEGES;
```

启动的时候记得加映射

```bash
docker run -p 3306:3306 mysql
```



#### 2. 试试连接

映射的端口是 3306，可以用 hostname -I 看IP，别忘记啦~
