# 在Ubuntu下部署Nginx

### Nginx 官方文档：https://nginx.org/en/docs/



在安装任何环境之前，都要记得首先把软件包都更新一下

```
sudo apt-get update
sudo apt-get upgrade
```

若需要执行更新，命令行会提示，按需输入即可！



```
apt list --upgradable
```

此命令可以查看哪些软件包需要更新



#### 而后就可以开始部署Nginx了

### 1. 安装 Nginx

```
apt install nginx
```

安装软件包



```
sudo systemctl start nginx
sudo systemctl enable nginx
```

安装完成后，启动 Nginx 服务并设置开机自启动



```
sudo systemctl reload nginx
```

重载 Nginx 的配置

### 2. 配置防火墙

如果启用了防火墙，需要允许 HTTP 和 HTTPS 流量通过：

```
sudo ufw allow 'Nginx HTTP'
sudo ufw allow 'Nginx HTTPS'
```



**截至到这里，就可以使用 IP地址 与 80 端口号进入页面啦！**

### 4. 创建站点页面（静态页面）

在 /var/www/ 下创建一个属于自己的静态页面目录吧！

```
root@qianyue-VMware-Virtual-Platform:/var/www# mkdir spring
```

创建一个 **index.html** !

```
cd spring
vim indexl.html
```

随便写点！

```
<!DOCTYPE html>
<html>
        <head>
        		<meta charset="UTF-8">
                <title>SpringAurora's Website</title>
        </head>
        <body>
                <h1>欢迎来到春月的测试页面！</h1>
                <p>SpringAurora running on Nginx Web Server - On Ubuntu 22.04</p>
        </body>
</html>
```

记得保存！

```
:wq
```



然后回到配置文件目录创建一个属于自己站点的配置文件（/etc/nginx/sites-available）

```
cd /etc/nginx/sites-available
vim spring
```

```
server {
listen 801;
root /var/www/spring;
index index.html;
}
```

记得保存！

因为 default 默认站点占用了 80 端口，那我们创建一个访问 801 端口的站点吧！

创建完站点文件后，记得创建符号链接到 `sites-enabled` 目录，以便启用

```
ln -s /etc/nginx/sites-available/spring /etc/nginx/sites-enabled/
```

以及不要忘了放行 801 端口，因为 Nginx 默认只放行 80 端口！

```
ufw allow 801
systemctl restart nginx
```

放行部署完后，重启一下 Nginx 服务，就可以访问页面啦！

```
ip addr show
```

别忘了怎么查看ip！
