# 配置FTP

~~才不是为了传文件方便才部署的FTP服务器呢，才不是呢~~



### 1. 安装FTP

照例，首先把软件包都更新一下吧

```
sudo apt-get update
sudo apt-get upgrade
```

安装FTP

```
apt-get install vsftpd
```

别忘了启动服务、也可以设置一下开机自启！

```
systemctl start vsftpd
systemctl enable vsftpd
```

vsftpd 的配置文件通常位于 `/etc/vsftpd.conf`。可以编辑这个文件来根据需要调整 FTP 服务器的配置。

确保 `local_enable` 设置为 `YES`，这样本地系统用户才能登录到 FTP 服务器。

如果希望只允许经过身份验证的用户访问，可以禁用匿名访问。确保 `anonymous_enable` 设置为 `NO`。

`write_enable` 的注释符号记得去掉，不然文件只能看不能上传。



### 2. 配置防火墙

记得放行21端口！

```
ufw allow 21/tcp
```



### 3. 创建FTP专有用户

创建一个名字为 ftpqy 的用户

```
adduser ftpqy
```



确认 FTP 用户有权限访问其主目录。检查用户的主目录权限，确保其可以读取和写入

顺带着也算是在 home 下创建了一个属于 FTP 用户的个人文件夹

```
sudo chown ftpuser:ftpuser /home/ftpqy
sudo chmod 755 /home/ftpqy
```



创建完后记得 reload 一下 FTP服务，就可以连接使用啦！

```
systemctl reload vsftpd
```
