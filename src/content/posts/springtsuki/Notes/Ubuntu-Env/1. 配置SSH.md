# 配置SSH

如果想利用VScode连接SSH达成在Ubuntu平台下实现开发环境，还是得做点操作的



### 1. Ubuntu安装SSH

那么首先把软件包都更新一下吧

```
sudo apt-get update
sudo apt-get upgrade
```

更新完毕后，就可以安装SSH服务器与客户端了

```
sudo apt install openssh-server
sudo apt install openssh-client
```

记得关闭防火墙，否则会导致远程计算机无法连接到此SSH客户端

```
sudo ufw disable
```



### 2. 配置SSH

配置ssh_config

```
sudo vi /etc/ssh/ssh_config
```

把此行的注释符号删去

```
#   PasswordAuthentication yes
```

更新为

```
    PasswordAuthentication yes
```



配置sshd_config

```
sudo vi /etc/ssh/sshd_config
```

并在# PermitRootLogin prohibit-password之后添加一行：PermitRootlogin yes

```
#PermitRootLogin prohibit-password
#此处新增
PermitRootLogin yes
```



那么截止到此处SSH就配置完毕了



### 3. 常用的SSH工具

Windows 其实本身就自带了 SSH 工具哦，只不过需要在 **Windows 设置下 -> 系统 -> 可选设置 中，启用 OpenSSH 功能 **即可！

其他软件类的话，例如 Windows 下就可以使用 **Putty** ！
