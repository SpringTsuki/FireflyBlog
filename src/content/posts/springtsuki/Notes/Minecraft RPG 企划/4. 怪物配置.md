# 4. 怪物配置

怪物分为三种类型，被动型、中立型与主动型

```
绿色：被动型 - 不会主动攻击玩家
黄色：中立型 - 被玩家攻击后会主动攻击玩家
红色：主动型 - 会主动攻击玩家
```



基本代码配置：

全局配置：

```
实体共通标签：
渲染实体名称：渲染 | CustomNameVisible:true
实体名：自定义 | CustomName:{translate:"xxx"}
实体标签：自定义 | Tags:[""]
实体状态：自定义 | attributes:[]

生物共通标签:
实体属性：自定义 | attributes:[]

AI生物共通标签：
是否可捡起掉落物：不可 | CanPickUpLoot:false
```

怪物全局配置：

```
怪物 Tag | Tags:["monster"]
```

```
装备：自定义 | equipment:{}
被击杀后战利品表：自定义 | DeathLootTable:"springrpg:"
```

村民配置：

```
村民 Tag | Tags:["villager"]
```

```
实体抵抗伤害：抵抗 | Invulnerable:true
实体属性配置：0 移动速度、50 血量 | attributes:[{id:"movement_speed",base:0},{id:"movement_speed",base:0}]
```
