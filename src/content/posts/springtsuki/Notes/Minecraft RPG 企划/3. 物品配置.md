# 3. 物品配置

针对于 武器、防具、药品、杂货 都可以划分不同的等阶~

```
一阶：黄绿 - 稀有度：常见
二阶：淡蓝 - 稀有度：罕见
三阶：紫色 - 稀有度：珍贵
四阶：橙色 - 稀有度：史诗
五阶：红色 - 稀有度：传说
六阶（神器）：粉红 - 稀有度：神话
```

获取难度

```
大世界购买：金币购买 | 材料 + 金币 合成
大世界掉落形式：部分小怪稀有掉落

副本形式：
普通难度：可以获取四阶装备或五阶装备
高难难度：获得六阶装备（或六阶装备低保）或五阶装备
```

武器、防具、药品 物品格式：

```
物品名称
物品稀有度
物品介绍
```

物品配置：

```
物品 Tag | Tags:["item"]
```

武器机制

```
近战：剑、斧
远程：弓、弩

无魔法攻击手段，均为物理攻击（倾向原版CTM）
```

```
/give @p minecraft:wooden_sword[attribute_modifiers=[{id:"attack_damage",type:"minecraft:attack_damage",amount:1,operation:add_value,slot:mainhand},{id:"attack_speed",type:"minecraft:attack_speed",amount:-3,operation:add_value,slot:mainhand}],custom_name={translate:"wooden_sword.newer_name",italic:false},lore=[{translate:"rare_common"},{translate:"wooden_sword.newer_detail",italic:false}]]
```
