# Vim的安装和配置

那么Vim在Ubuntu中默认是没有的，所以需要安装一个

（确切来讲是安装了vim-command，但是不太好用，删了）

```
sudo apt remove vim-command
sudo apt install vim
```



然后输入vim就可以使用了

```
vim
```



#### 那么vim的一些基础操作不要忘记如何使用

```
:q
```

退出当前的文本文件



```
:qa
```

退出文本文件并放弃更改



```
:wq
```

保存退出



```
:wq!
```

强制保存并退出



```
i
```

按下i健即可插入开始编写文档
