# 提权（获取root）

在Ubuntu中root默认是没有密码的，所以需要使用sudo权限并使用passwd将root重设密码

```
sudo passwd root
```

而后再使用su提权即可

```
su root
```

别忘了用完之后退出root

```
exit
```



当然，如果你想进一步配置sudo指令免密操作的话，得先安装个文本编辑器（vim）

安装与使用方法看第六页面，安装完了再回到这里吧！



#### 配置操作如下

由于sudoers是只读文件，所以需要chmod配置一下权限

```
chmod 770 /etc/sudoers
```

而后使用文本编辑器打开

```
vim /etc/sudoers
```

将此处的代码进行更新

```
# Allow members of group sudo to execute any command
%sudo   ALL=(ALL:ALL) ALL
```

更新为

```
# Allow members of group sudo to execute any command
%sudo   ALL=NOPASSWD:ALL
```

而后别忘记把权限改回，改回后重启即可应用配置

```
chmod 440 /etc/sudoers
reboot
```

使用sudo -s也可以快速切换到root用户了

```
sudo -s
```
