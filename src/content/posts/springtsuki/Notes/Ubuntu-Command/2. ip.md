# 关于ip指令的详解

在Ubuntu中查看网络配置与IP地址，则需要用到ip指令



```cmd
ip addr show
```

查看当前网络的IP地址、示例输出如下：

```bash
qianyue@qianyue-VMware-Virtual-Platform:~/桌面$ ip addr show
1: lo: <LOOPBACK,UP,LOWER_UP> mtu 65536 qdisc noqueue state UNKNOWN group default qlen 1000
    link/loopback 00:00:00:00:00:00 brd 00:00:00:00:00:00
    inet 127.0.0.1/8 scope host lo
       valid_lft forever preferred_lft forever
    inet6 ::1/128 scope host noprefixroute 
       valid_lft forever preferred_lft forever
2: ens33: <BROADCAST,MULTICAST,UP,LOWER_UP> mtu 1500 qdisc fq_codel state UP group default qlen 1000
    link/ether 00:0c:29:2b:14:4f brd ff:ff:ff:ff:ff:ff
    altname enp2s1
    inet 192.168.31.197/24 brd 192.168.31.255 scope global dynamic noprefixroute ens33
       valid_lft 43100sec preferred_lft 43100sec
    inet6 fe80::cca3:725:e059:a1a3/64 scope link noprefixroute 
       valid_lft forever preferred_lft forever

```

即可查看到当前配置的ens33网卡下，inet的ip地址为：192.168.31.197



当然，hostname也可以直接查询

```bash
hostname -I
```

示例输出如下：

```
192.168.31.197 
```

若需要虚拟机在当前局域网下分配独立IP，需要将VMware的该Ubuntu虚拟机设置为桥接模式

再查询IP后，使用此IP即可ping通
