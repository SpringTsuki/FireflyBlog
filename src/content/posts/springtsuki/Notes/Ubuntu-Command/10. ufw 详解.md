# ufw详解

`UFW`（Uncomplicated Firewall）是一个简化的防火墙管理工具，用于在 Linux 系统上管理 `iptables` 防火墙规则。它提供了一个简单的命令行界面，允许你快速配置和管理防火墙规则。

#### 

#### 1. 安装 ufw

在部分 Linux 发行版上可能原生不自带 ufw。用软件包管理器执行更新后安装即可。



#### 2. 启用与禁用 ufw

```
ufw enable
ufw disable
```

就这么简单



#### 3. 查看防火墙状态

```
ufw status
```

如果希望看到更加详细的参数，可以添加verbose

```
ufw status verbose
```



#### 4. 放行指定端口

放行一个端口号为80的端口

```
ufw allow 80
```

放行一个端口号为80且为TCP协议的端口

```
ufw allow 80/tcp
```

拒绝80端口的访问

```
ufw deny 80
```



#### 5. 允许一个IP的访问

例如允许本机IP访问，这样设定后这个IP将可以访问所有端口

```
ufw allow from 192.168.31.72
```

当然，也可以指定IP访问指定的端口

```
ufw allow from 192.168.31.72 to any port 22
```

进一步的，也可以指定IP访问指定的端口且遵循指定的协议

```
ufw allow from 192.168.31.72 to any port 22 proto tcp
```



#### 6. 管理规则

得先查看目前ufw有多少规则吧？

```
ufw status numbered
```

删除编号为1的规则

```
ufw delete 1
```



#### 7. 放行服务规则

ufw很方便的可以直接放行规则，例如允许 SSH 服务穿过防火墙

```
ufw allow ssh
```

或者是允许 TCP？

```
ufw allow tcp
```
