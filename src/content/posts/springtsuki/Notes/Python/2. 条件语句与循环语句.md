##### 2. 循环语句（while）

```python
count = 0
while count < 9:
    print('The count is:', count)
    count = count + 1

print("While End")
```

一个很简单的 while 循环，当然，也可以使用 break 跳出循环

```Python
count = 0
while count < 9:
    print('The count is:', count)
    count = count + 1
    if count == 5:
        break

print("While End")
```

当然，也可以使用 continue 跳过一次循环



##### 3. 循环语句（for）

```python
for letter in 'Python':  # 第一个实例
    print("当前字母: %s" % letter)

fruits = ['banana', 'apple', 'mango']
for fruit in fruits:  # 第二个实例
    print('当前水果: %s' % fruit)

print("Good bye!")
```

当然，for 中也是可以使用 break 与 continue 的
