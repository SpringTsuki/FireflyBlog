# 列表

最基本的数据结构（类似于 C 的数组）

#### 1. 获取元素

同样的，也可以使用相同的操作获取列表里的元素

```python
list1 = ['one', 'two', 'three']
list2 = [1, 2, 3, 4, 5]

print(list1)
print(list1[2])
print(list1[0:2])

print(list1 + list2)
```

运行结果：

```
['one', 'two', 'three']
three
['one', 'two']
['one', 'two', 'three', 1, 2, 3, 4, 5]
```



使用 for 循环将列表里的元素转化为 str ，并写入 var 并输出呢 ...?

```python
list1 = ['one', 'two', 'three']
list2 = [1, 2, 3, 4, 5]

var1 = ""
var2 = ""
for var in list1:
    var1 += str(var)

for var in list2:
    var2 += str(var)

print(var1 + "\n" + var2)
```

运行结果：

```
onetwothree
12345
```
