# Java的开始

不管怎么样，先从默认生成的Hello World开始吧！

```Java
public class Main {
    public static void main(String[] args) {
        System.out.print("Hello and welcome!\n");
        for (int i = 1; i <= 5; i++) {
            System.out.println("i = " + i);
        }
    }
}
```



很简单，首先新建了一个类（clase）命名为main

而后在此类下创建了一个方法 ~~函数~~ main（）

public 表示此方法的 **访问修饰符**

static 表示此方法的 **关键字**

void 表示此方法的 **返回类型**

main 表示此方法的 **名字**

内部的 **String[]** 含义为：此是一个 String 类（字符串类）

**args** 则为 **字符串数组** 的含义

这里也顺带着介绍一下Java的版本如何通过代码输出

```Java
String javaVersion = System.getProperty("java.version");
System.out.println("Java Version: " + javaVersion);
```

使用 System.getProperty(“java.version”) 即可获取Java版本

当然，使用命令行终端的话，只需要java -version即可

```
java -version
```
