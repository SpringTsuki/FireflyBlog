# 对象与类

敲黑板，这里是重点。（https://www.runoob.com/java/java-object-classes.html - 菜鸟教程 - 对象与类）

#### 1. 类

为定义对象的蓝图、包括属性与方法



#### 2. 对象

类的实例，具有状态与行为



#### 3. 继承

通常一个类可以一个继承另一个类的方法



#### 4. 封装

将对象的状态（字段）私有化，通过公共方法访问。

就像这样：

```java
private String name; 
public String getName() { return name; }
```



#### 5. 多态（有点难理解）

对象可以表现为多种形态，主要通过方法重载和方法重写实现。

示例：

方法重载：

```java
public int add(int a, int b) { ... }
```

```java
public double add(double a, double b) { ... }
```

方法重写：

```java
@Override public void makeSound() {
    System.out.println("Meow"); }
```



#### **6、抽象（Abstraction）**：

使用抽象类和接口来定义必须实现的方法，不提供具体实现。

示例：

抽象类：

```java
public abstract class Shape { abstract void draw(); }
```

接口：

```java
public interface Animal { void eat(); }
```



#### **7、接口（Interface）**：

定义类必须实现的方法，支持多重继承。

示例：

```java
public interface Drivable { void drive(); }
```



#### **8、方法（Method）**：

定义类的行为，包含在类中的函数。

示例：

```java
public void displayInfo() { System.out.println("Info"); }
```



#### **9、方法重载（Method Overloading）**：

同一个类中可以有多个同名的方法，但参数不同。

示例：

```java
public class MathUtils {
    public int add(int a, int b) {
        return a + b;
    }

    public double add(double a, double b) {
        return a + b;
    }
}
```



那么一大堆类与对象的特性都看完了，想必也肯定记不住几个。

放心啦，之后都会再次用到并学习这些的。

当然，最重要的部分，其实是要理解 **类 与 对象 的真正含义。**

简单来说，类即模板，而对象可以理解为套用对象的具体状态。而且，一个类可以有多个方法。

```java
public class Cat {
    String breed;
    int size;
    String colour;
    int age;
 
    void eat() {
    }
 
    void run() {
    }
 
    void sleep(){
    }
 
    void name(){
    }
}
```

例如这里定义了一个 **猫（Cat）** 的 **类（class）**

而后面的 String breed、int size、String color、int age 则为这个 类 的 **属性**

在后续的使用时，可以通过 **get 与 set 方法**来访问或修改 **类的成员（即 对象 - Object）**的属性



#### 那么接下来实战看看吧！

在 `cat.java` 中，定义了一个名为 `cat` 的 `class`，而后定义了它的属性，分别为 `age`、`name`、`cute`

其中 `age` 为 `int` 型变量、`name` 为 `String` 型变量、`cute` 为 `Boolean` 型变量

同时，又定义了一个方法，方法内容为输出定义完成的属性



`cat.java` 如下：

```java
package day1;

public class cat {
    int age;
    String name;
    boolean cute;

    void catinfo(){
        System.out.println("猫猫的名字是:" + name);
        System.out.println("猫猫的年龄是:" + age);
        System.out.println("猫猫是否可爱:");
        if (cute) {
            System.out.println("可爱!");
        } else {
            System.out.println("呜呜!");
        }
    }
}
```

而后，在 `Main .java` 中创建 `cat` 的对象并进行了属性的修改，同时调用对象（类）里的方法

`Main.java` 如下：

```java
package day1;

public class Main {
    public static void main(String[] args) {
        cat cat_1 = new cat();
        cat cat_2 = new cat();

        //可以通过此种方法=对属性进行直接访问并修改
        cat_1.age = 5;
        cat_1.name = "Firefly";
        cat_1.cute = true;

        cat_1.catinfo();
    }
}
```

运行 `Main.java` 输出的结果为：

```
猫猫的名字是:Firefly
猫猫的年龄是:5
猫猫是否可爱:
可爱!
```
