```bash

```



#### 添加远程桌面服务

服务器管理器 -> 添加角色与功能 -> 下一步 -> 基于角色或基于功能的安装 -> 下一步

勾选：远程桌面服务 -> 勾选：远程桌面会话服务

安装服务后重启即可



#### 阻止 NTLM 解决方案

禁用 NTLM ：

本地组策略编辑器（gpedit.msc）：计算机配置 -> Windows设置 -> 安全设置 -> 本地策略编辑器 -> 安全选项

网络安全：

LAN 管理器身份验证级别 -> 仅发送 NTLMv2响应 -> 拒绝 LM 和 NTLM



修改远程桌面服务：

本地组策略编辑器（gpedit.msc）：计算机配置 -> 管理模板 -> Windows组件 -> 远程桌面服务 -> 安全 -> 远程(RDP)连接要求使用指定的软件层 -> 启用/修改为 RDP 保存



Oracle 修正：**（如果没有此选项记得更新Windows！！！）**

本地组策略编辑器（gpedit.msc）：计算机配置 -> 管理模板 -> 系统 -> 用户凭据 -> Oracle加密 -> 易受攻击



更新策略：

```
gpupdate
```
