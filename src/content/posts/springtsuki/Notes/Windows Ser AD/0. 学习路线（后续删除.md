### 学习路线

1. **安全和权限管理**
   - **委派权限**：如何委派权限给其他管理员。
   - **安全性**：理解并配置 Kerberos 认证，NTLM，如何设置账户策略、密码策略和锁定策略。
   - **多重身份验证 (MFA)** 和 **条件访问**。
2. **故障排除**
   - **诊断 AD 问题**：使用如 `dcdiag`、`repadmin`、`netdiag` 等工具进行故障排除。
   - **Active Directory 数据恢复**：了解备份和恢复策略。
3. **高级功能**
   - **Active Directory 联合身份认证 (ADFS)**：了解如何实现跨域单点登录（SSO）。
   - **Active Directory 权限管理和审计**：设置高级安全性审计，监控和管理用户活动。
4. **脚本与自动化**
   - 学习如何使用 PowerShell 管理 AD，以实现自动化。
