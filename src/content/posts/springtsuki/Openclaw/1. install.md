## 1. 安装

Github：https://github.com/openclaw/openclaw

安装：
```bash
npm install -g openclaw@latest
# or: pnpm add -g openclaw@latest

openclaw onboard --install-daemon
```

## 2. 配置
WebUI 启用方式：
```bash
ssh -N -L 18789:127.0.0.1:18789 qianyue@192.168.31.29
```

由于 OpenClaw 设定了很脑残的仅 HTTPS 或 Localhost 访问，所以局域网主机IP加端口则会直接提示 HTTPS 异常。建议还是通过转发的方式进行访问。

这里也需要修改 SecureCRT 的配置，才可以让端口转发出来，让局域网的其他设备收到转发请求，通过本机的 Localhost 进行访问。

设置端口转发的目标 IP 与 本机 IP 均为 127.0.0.1，而后填写 OpenClaw 的端口号：18789，即可。

## 3. 获取Token
```bash
cat ~/.openclaw/openclaw.json
```
获取 Token 就可以访问 WebUI 进行配置了
