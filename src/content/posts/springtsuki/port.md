1. SSH - 22 - TCP - 远程登陆
2. FTP - 21 - TCP - 文件传输
3. SFTP - 22 - TCP - 基于SSH的文件传输
4. SMTP - 25 - TCP - 邮件发送
5. POP3 - 110 - TCP - 邮件接收
6. HTTP - 80 - TCP - 常规Web服务
7. HTTPS - 443 - TCP - 加密Web服务
8. DNS - 53 - TCP/UDP - 域名解析
9. DHCP - 67/68 - UDP - 动态IP分配
10. MySQL - 3306 - TCP - 数据库服务
11. Redis - 6379 - TCP - 缓存服务
12. Nginx - 80/443 - Web端口
13. Apache - 80/443 - Web端口
14. Astrobot WebUI - 6185 - 管理面板
15. Astrobot Onebot WS - 6199 - Onebot Websocket 服务
16. NapCat WebUI - 6099 - 管理面板
